## 📌 What We're Building in Phase 1

A Python program that:
1. Wakes up every morning
2. Reads top AI news from the internet
3. Picks the most interesting story
4. Uses an AI (Gemini) to write a clean, Instagram-style summary
5. Saves that post to a database (Supabase)

No front-end yet. No pretty UI. Just the engine that powers the whole thing.
